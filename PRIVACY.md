# Privacy Policy for Session Sushi

**Last Updated:** January 2025

## Data Collection

Session Sushi does **NOT** collect or store any data on external servers.

## What the Extension Does

Session Sushi is a local tool that:

1. **Accesses browser cookies** - reads and manages cookies stored in your browser
2. **Stores data locally** - saves OAuth tokens and preferences in your browser's local storage
3. **Communicates with Microsoft** - sends API requests to Microsoft's servers when you use OAuth or Graph API features

## Data Storage

All data remains on your device:

- OAuth tokens are stored in browser local storage
- No data is sent to the extension developer
- No analytics or telemetry
- No third-party tracking

## Permissions

The extension requires browser permissions to:

- Read and modify cookies
- Access local storage
- Make requests to Microsoft APIs
- Download exported data files
